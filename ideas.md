## アイデア1: 「Blueprint（設計図）」スタイル

### Design Movement
インダストリアル・ブループリント × テクニカル・ミニマリズム

### Core Principles
1. **精密さと信頼性**: 設計図のような精密なグリッドと線で「方程式」「型」を視覚化
2. **構造の可視化**: 営業プロセスを工学的・システマティックに表現
3. **ダークモード基調**: 深いネイビー/ブラックに白い線とアクセントカラー
4. **数式・図解の多用**: 方程式という名前に合わせた数学的ビジュアル

### Color Philosophy
- **Primary**: ディープネイビー (#0A1628) - 信頼性と専門性
- **Secondary**: ブループリント・ブルー (#2563EB) - テクニカルな印象
- **Accent**: エレクトリック・シアン (#06B6D4) - CTAと重要ポイント
- **Text**: ピュアホワイト (#FFFFFF) と ライトグレー (#94A3B8)

### Layout Paradigm
- 斜めのグリッドライン（15度傾斜）がセクション間を繋ぐ
- 左右非対称レイアウト（テキスト60% / ビジュアル40%）
- フローティングカード要素が「設計図上のパーツ」のように配置

### Signature Elements
1. アニメーションする点線（設計図の寸法線風）
2. 数式風のタイポグラフィ（成約 = 信頼 × 提案力 × タイミング）
3. ステップ間を繋ぐコネクターライン

### Interaction Philosophy
- ホバーで要素が「選択された設計パーツ」のようにハイライト
- スクロールで設計図が「組み上がっていく」アニメーション
- CTAボタンは「実行」「起動」を想起させるデザイン

### Animation
- セクション進入時に線が描かれるドローイングアニメーション
- 数字カウントアップ（実績数値）
- パララックススクロールで奥行き感

### Typography System
- **見出し**: Space Grotesk（幾何学的・テクニカル）
- **本文**: Noto Sans JP（可読性重視）
- **アクセント**: JetBrains Mono（数式・コード風要素）

2. **余白の美学**: 日本的な「間」を活かした洗練されたレイアウト

### Color Philosophy
- **Primary**: オフホワイト (#FAFAF9) - 清潔感と高級感
- **Secondary**: チャコール (#1C1917) - 力強さと信頼
- **Accent**: ゴールド (#D4AF37) - 成功と価値

- 大胆な余白（セクション間に十分なブレス）

- **本文**: Noto Sans JP（現代的・可読性）

## 選択したデザイン: アイデア2「Gold Standard（黄金基準）」スタイル

### 選択理由
1. **ターゲットとの親和性**: 経営者・フリーランス・制作者は「高級感」「信頼性」を重視する傾向
2. **差別化**: 多くの営業講座LPがビビッドカラーやグラデーションを使う中、落ち着いた高級感で差別化
3. **日本語の美しさ**: 縦書き要素や和のテイストで日本市場に最適化
4. **「方程式」との相性**: ゴールドは「黄金比」「黄金律」を連想させ、「成約の方程式」というネーミングと調和
5. **長期的なブランド構築**: 派手さより品格で、長く使えるデザイン

### 実装方針
- ライトモード基調（オフホワイト背景）
- ゴールドアクセントを効果的に配置
- 余白を十分に取り、情報の優先度を明確に
- 高品質な画像生成でビジュアルインパクトを確保
